Background Images for Momentum Pro

Instructions:

  1. For each image below, right-click on the image and select "Save Image As..."
  2. Save each image with the filename shown below the image
  3. Save all images to the src/assets/images folder in your project
  4. After downloading all images, you can delete this HTML file

4. Geometric Abstract

Save as: bg-geometric.jpg

After Downloading:

Once you've downloaded all images to the correct folder, you can test the background by:

  1. Opening the App.vue file
  2. Uncommenting the background image you want to use
  3. Adjusting the opacity value to your preference (between 0.05 and 0.3)
  4. Running the app to see the result
