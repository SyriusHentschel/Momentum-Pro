MIME Type Information

JavaScript modules require the correct MIME type: application/javascript

If you see errors in the console about MIME types, it means the server is not correctly configured.

Headers Test

The following headers should be set for JavaScript files:

Content-Type: application/javascript
Cache-Control: public, max-age=31536000, immutable
